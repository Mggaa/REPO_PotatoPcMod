## REPO Potato Performance

A performance-focused mod made for low-end PCs.

This mod disables heavy visual effects that cause stutter and low FPS, especially on systems without a dedicated GPU.

### Features
- Disable post-processing effects (color grading, film grain, vignette, bloom, etc)
- Disable or reduce shadows
- Reduce overall visual cost
- All options configurable via REPOConfig
- Can be enabled/disabled at runtime

### Recommended For
- Low-end PCs
- Integrated graphics
- Laptops
- Players experiencing stutter even with high FPS

### Dependencies
- BepInEx
- REPOConfig
- MenuLib

### Notes
- Some settings are reapplied on scene load to prevent the game from restoring effects.
- Visual quality will be reduced to improve performance.

Created to help players with low-end hardware.
